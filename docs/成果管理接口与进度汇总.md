## 一、功能范围
- 成果管理（论文/项目）
  - 列表、详情、创建、更新、删除（软删）
  - 发布/取消发布、审核/取消审核
- 作者管理（成果下作者）
  - 列表、添加、更新、删除（软删）

- 字典

## 三、权限策略
- 管理员：所有操作
- 拥有者（owner_user_id=本人）：可编辑/删除自己的成果；可管理该成果的作者
- 作者可见性：仅内部作者有效；对外/个人页展示时会过滤 visible=false 的作者
- 发布/审核：仅管理员

## 四、数据与校验
- 论文（type=1）：paperType、publishDate 必填；禁止传 project* 字段
- 项目（type=2）：projectType、projectStartDate 必填；publishDate 必须为空；projectEndDate 可空或 >= projectStartDate
- fundingAmount >=0（若传）
- 作者顺序（authorOrder）在同一成果内唯一，>0
- 外部作者：userId=null 且 name 必填

## 五、删除策略
- 成果：软删（deleted=true，@TableLogic），查询默认过滤
- 作者：软删（deleted=true，@TableLogic），删除成果时联动软删作者
